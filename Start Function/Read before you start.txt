NOT:Çalıştırma esnasında yaşanan bir hatadan veri tabanı sorumludur(bazen bağlanmama durumları oluyor).



Start fonksiyonunu çalıştırmak için:

=>GET localhost:4000/buy/type/amount
örn: GET localhost:4000/buy/latte/1

=>GET localhost:4000/take

=>GET localhost:4000/remaining

=>GET localhost:4000/fill/type/amount
örn: GET localhost:4000/fill/water/100
